# Aedn Technologies

We make **NetDoc Pro** — local-first network documentation software for Windows.

## What NetDoc Pro does

Point it at a subnet and it discovers your network, draws a topology diagram with verified port-level adjacency from SNMP / LLDP / CDP, parses running-configs from twelve vendors (Cisco, Juniper, Dell, HP, Aruba, Huawei, MikroTik, FortiGate, Extreme, Brocade), detects operating systems via SSH / SNMP / WinRM, checks every device for known CVEs, and generates branded HTML and PDF reports.

Built for MSPs, in-house IT teams, and network engineers who need professional documentation without sending their network data to someone else's cloud.

- All data stays on your machine
- Credentials encrypted with Windows DPAPI
- Diagrams stored as portable `.nsdoc` files (JSON, no vendor lock-in)
- Only Telemetry for cras reports on Sentry and Licensing, no usage tracking
- Free tier available; Pro and Business tiers for full discovery, automation, and semi-team collaboration
